# Manyana_deploy
Manyana机器人一键部署脚本，适用于带有apt和yum的系统 :yum: 

使用方法


`wget https://gitee.com/laixi_lingdun/Manyana_deploy/raw/main/install.sh`

`chmod 777 install.sh`

`./install.sh`

 **_脚本有严格混淆，运行时要多等一会。_**
